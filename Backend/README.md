# Stock Market App Backend

This is the FastAPI backend for the AI-powered Stock Market Application.

## Deployment on Hugging Face Spaces

1. Create a new **Space** on Hugging Face.
2. Select **Docker** as the SDK.
3. Upload the contents of the `Backend` folder.
4. Add your `GROQ_API_KEY` and any other secrets in the **Settings > Variables and secrets** section of your Space.
5. The app will automatically build and start on port 7860.
